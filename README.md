# Sales Prediction Project

This project uses Machine Learning to predict sales based on advertising data.

## Overview
The model analyzes the relationship between advertising channels and sales performance.

## Dataset
- TV Advertising
- Radio Advertising
- Newspaper Advertising
- Sales (Target variable)

## Models
- Linear Regression
- Random Forest Regressor

## Workflow
- Data preprocessing
- Exploratory data analysis
- Model training
- Evaluation
- Visualization of results

## Output Files
- Correlation heatmap
- Prediction vs Actual plot
- Residual analysis plot
- Trained model file (.pkl)

## How to Run
```bash
python sales_prediction.py
